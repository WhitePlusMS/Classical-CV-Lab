01 · 创意介绍

把"看不懂的公式"变成"摸得到的光圈"

传统计算机视觉课堂最大的难题：算法都写在公式里，学生只能死记硬背， 却看不到参数变化时图像到底发生了什么。视界实验室用一整套可交互的实验台， 让每个算法都变成可观察、可调参、可推演的"光学暗房"。

想解决的问题

课堂里"看得见摸不着"的 CV 算法

  • 卷积核、阈值、高斯金字塔这些概念只存在于板书与公式里，调一个参数会发生什么全靠脑补。
  • OpenCV 调用一行代码就出结果，学生只学到了"调用"，没学到"原理"。
  • 课后想验证一个想法，要先装环境、配 Python、装依赖，门槛高到劝退。
  • 复杂流程（SIFT、Canny、OTSU）只有最终图，看不到中间每一步的中间结果。
我们的产品形态

一个打开浏览器就能用的"CV 实验台"

  • 纯前端网站（Next.js），无需安装任何环境，点开链接就进入实验。
  • 31 个交互式概念页，覆盖预处理、标定、校正、目标检测全链路。
  • 每个算法都用 TypeScript 从零实现，不依赖 OpenCV，源码可读可改。
  • 滑动条实时调参，像素级查看灰度值、梯度、邻域，复杂流程分步可视化。
02 · 目标用户与痛点

为"想真正理解 CV"的人而做

不论是第一次接触图像处理的学生，还是想把课堂讲"活"的老师， 亦或是需要快速回顾某个算法细节的工程师——都能在这里找到自己需要的那个实验。

计算机视觉课程学生

// 课前预习 · 课后复习

预习时看不懂公式推导，复习时想换个参数验证理解。视界实验室让每个概念都能"动手试一试"，把抽象的数学关系变成可见的图像变化，建立真正的直觉。

CV / 图像处理教师

// 课堂演示 · 实验作业

课堂上讲到 Canny、OTSU，切到对应页面拖动阈值，学生立刻看到边缘如何变化。可作为课堂演示工具，也可作为开放实验作业平台，省去自建实验环境的精力。

算法工程师 / 自学者

// 算法回顾 · 原理深挖

用惯了 OpenCV 调用，偶尔需要回忆 SIFT 描述子到底怎么生成、HOG 的 cell/block 怎么划分。这里提供从零实现的源码与可视化，是原理深挖与快速回顾的便携手册。

03 · 价值与意义

让"理解原理"不再依赖环境与天赋

从效率提升、教育普惠到工程示范三个角度，视界实验室试图把传统 CV 教育中 "看不见、摸不着、装不上"的三重门槛一次性拉平。

VALUE · 01

效率提升

把"装环境 → 写代码 → 调参数 → 看结果"的传统闭环压缩成"打开网页 → 拖滑块 → 看变化"。一次实验从几十分钟缩短到几秒，让学习时间真正花在理解原理上，而非与依赖、版本、环境作斗争。

VALUE · 02

教育普惠

无需高性能电脑、无需安装配置，任何一台能上网的设备都能进入完整的 CV 实验室。降低硬件与门槛壁垒，让偏远地区学生、自学者也能获得与重点高校同等质量的"可动手"教学资源。

VALUE · 03

工程示范

所有算法以纯 TypeScript 从零实现并开源，本身即是一份"如何在浏览器里写好数值计算"的教学范本。展示前端不仅能做 UI，也能承担严肃的算法可视化与教学职责，为同类教具提供可复用范式。

04 · 功能亮点

五个让算法"活起来"的设计

交互式参数调优

每个概念页配备参数面板，滑动条 / 下拉选择 / 卷积核编辑器即时驱动算法重新计算，改一个数图像立刻变。

像素级导航

方向键 + 点击可定位任意像素，查看灰度值、梯度、邻域窗口等细节，把"图像即矩阵"这句话变成可触摸的现实。

步骤级可视化

复杂算法（SIFT、Canny、OTSU）拆解为多步骤流水线，每一步都展示中间计算结果，看清楚算法"如何一步步走到答案"。

纯前端零依赖

所有 CV 算法（卷积、形态学、SIFT、HOG 等）以纯 TypeScript 实现，无后端、无 OpenCV 运行时，部署即用。

统一教学组件体系

ConceptIntro、TeachingFlow、TeachingMath、TeachingPixel 等组件统一了"任务→思路→观察"的教学叙事，让 31 个页面风格一致、节奏可预期。

3D 可视化辅助

摄像机标定等空间概念借助 Three.js 提供三维场景，让内外参、畸变这些"想象不出"的几何关系变得直观可见。

05 · 技术架构

现代前端 + 从零实现的算法内核

用 Next.js 16 + React 19 构建交互层，所有 CV 算法以纯 TypeScript 在浏览器端实时运算。 架构上把"教学叙事组件"与"算法实现库"彻底分层，便于扩展与复用。
