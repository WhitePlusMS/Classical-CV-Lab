【标题】学习工作 | 视界实验室 · Classical CV Lab — 交互式计算机视觉学习实验台

【正文】

## 一、创意名称 + 创意介绍

**创意名称：视界实验室（Classical CV Lab）**

**想解决什么问题：**
传统计算机视觉课堂上，卷积、阈值、SIFT、HOG、张正友标定这些核心算法只存在于板书与公式里，学生调一个参数会发生什么全靠脑补；想动手验证，又得先装环境、配 Python、装 OpenCV，门槛高到劝退。算法"看得见摸不着"，原理学不透。

**为什么会想到做这个：**
我自己学 CV 时最痛苦的就是——公式背得滚瓜烂熟，却从没真正"看到"高斯金字塔一层层降采样是什么样、Canny 的双阈值如何决定边缘的生与死。调一次 OpenCV 一行代码出结果，但中间到底发生了什么完全是黑盒。于是想做一个让每个算法都"可观察、可调参、可推演"的浏览器实验台，把抽象的数学关系变成可见的图像变化。

**大概是什么产品：**
一个纯前端的交互式学习网站（Next.js + React + TypeScript），无需安装任何环境，打开链接就能进入 31 个计算机视觉概念实验页，拖动滑块实时看像素级变化，复杂算法分步骤可视化。

---

## 二、目标用户及痛点

**面向哪些用户：**
- 计算机视觉 / 图像处理课程的在校学生（预习与复习）
- CV 相关课程的高校教师（课堂演示与实验作业布置）
- 算法工程师与自学者（原理深挖与快速回顾）

**在什么场景下使用：**
- 课前预习看不懂公式推导、课后想换个参数验证理解时
- 课堂上讲到 Canny、OTSU，需要当场拖动阈值让学生看到边缘如何变化
- 工程师用惯了 OpenCV，偶尔需要回忆 SIFT 描述子如何生成、HOG 的 cell/block 如何划分

**当前痛点：**
如果没有这个产品，学生只能死记公式、脑补参数效果；教师要么自己在 PPT 里贴静态图，要么花大量时间自建 Python 实验环境；自学者想验证一个想法，装环境就要折腾半天。算法"看不见、摸不着、装不上"成为三重门槛。

---

## 三、价值与意义

**效率提升：** 把"装环境 → 写代码 → 调参数 → 看结果"的传统闭环压缩成"打开网页 → 拖滑块 → 看变化"，一次实验从几十分钟缩短到几秒，让学习时间真正花在理解原理上。

**教育普惠：** 无需高性能电脑、无需安装配置，任何一台能上网的设备都能进入完整的 CV 实验室，降低硬件与门槛壁垒，让偏远地区学生、自学者也能获得与重点高校同等质量的"可动手"教学资源。

**工程示范：** 所有 CV 算法以纯 TypeScript 从零实现、零 OpenCV 依赖并开源，本身即是一份"如何在浏览器里写好数值计算"的教学范本，为同类算法可视化教具提供可复用的范式。

### 项目核心亮点速览

- **31 个交互式概念页**，覆盖图像采集基础、预处理与几何校正、目标检测三大章节
- **纯 TypeScript 实现全部 CV 算法**（卷积、形态学、SIFT、SURF、HOG、Haar、张正友标定等），零 OpenCV 依赖
- **交互式参数调优**：滑动条 / 下拉选择 / 卷积核编辑器即时驱动算法重算
- **像素级导航**：方向键 + 点击定位任意像素，查看灰度值、梯度、邻域
- **步骤级可视化**：SIFT、Canny、OTSU 等复杂算法拆解为多步骤流水线，每步展示中间结果
- **3D 可视化辅助**：摄像机标定等空间概念借助 Three.js 三维呈现
- **统一教学组件体系**：ConceptIntro / TeachingFlow / TeachingMath / TeachingPixel，"任务→思路→观察"叙事一致

技术栈：Next.js 16 + React 19 + TypeScript 5 + Tailwind CSS 4 + Three.js
